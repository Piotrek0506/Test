# Fiszki — Projekt zaliczeniowy

Aplikacja webowa do nauki słówek w formie fiszek, napisana w czystym TypeScript. 
Użytkownik przechodzi przez talię, odsłania odpowiedzi, ocenia znajomość i na końcu otrzymuje podsumowanie.  
Postęp i wyniki zapisywane są w `localStorage`.

## Funkcje
- Wczytywanie talii z pliku JSON (min. 20 fiszek).  
- Ekrany: start, fiszka, podsumowanie.  
- Odsłanianie odpowiedzi (**Pokaż odpowiedź**).  
- Ocena: **Znam** / **Jeszcze nie**.  
- Nawigacja: **Poprzednia**, **Następna**, **Zakończ**.  
- Statystyki: licznik fiszek, czas per fiszka, łączny czas sesji.  
- Zapis postępu i wyników w `localStorage`.

## Technologie
- TypeScript 
- HTML / CSS  
- Live Server
